# Vaarta
Video Conferencing Application

This project is a video conferencing application built using Dart and Flutter for the frontend, and Firebase for the backend. It enables users to conduct in-person meetings and collaborate remotely through video calls.

## Features

- User Registration and Authentication: Users can create an account, log in, and manage their profile.
- Video Calls: Initiate or join video calls with other participants, enabling real-time communication and collaboration.
- Chat Functionality: Send instant messages during video calls to share links, notes, or important information.
- Screen Sharing: Share your screen with other participants to present slideshows, documents, or demonstrations.


## Technologies Used

- Dart: The programming language used for building mobile applications with Flutter.
- Flutter: A cross-platform framework for building beautiful native applications.
- Firebase: A cloud-based platform providing backend services, including authentication, real-time database, and cloud storage.
- Cloud Firestore: A flexible, scalable NoSQL database provided by Firebase.

## Getting Started

1. Clone the repository: `git clone https://github.com/your-username/video-conferencing-app.git`
2. Navigate to the project directory: `cd video-conferencing-app`
3. Ensure you have Flutter installed on your system: [Flutter installation guide](https://flutter.dev/docs/get-started/install)
4. Install the project dependencies: `flutter pub get`
5. Set up a Firebase project and enable necessary services (authentication, Firestore, etc.).
6. Configure the Firebase credentials in the Flutter project.
7. Run the application on an emulator or physical device: `flutter run`
